# Aetherland

**Fantasy MMORPG 2D na blockchainie Base**  
Prawdziwa własność postaci, przedmiotów i złota w otwartym świecie.

### O grze
Aetherland to przeglądarkowe 2D MMORPG w klimatach klasycznych tytułów jak Margonem, ale zbudowane od podstaw z myślą o Web3.

Dzięki technologii blockchain gracze **naprawdę posiadają** swoje postacie, ekwipunek, zasoby i osiągnięcia. Wszystko co zdobędziesz — możesz sprzedać, wymienić lub przenieść na inne konto.

### Główne cechy
- Postacie jako unikalne **NFT** (ERC-721) z poziomem, statystykami i historią
- Przedmioty jako **ERC-1155** (stackowalne)
- Wewnętrzna waluta **Aether** (ERC-20)
- Pełny player-driven economy — handel na wbudowanym marketplace
- Klasyczny gameplay: questy, eksploracja, crafting, PvE, PvP
- Hybrid architecture (on-chain = własność + ekonomia, off-chain = płynna rozgrywka)

### Technologie
- **Blockchain**: Base (Ethereum L2)
- **Frontend**: Phaser 3 + TypeScript
- **Smart Contracts**: Solidity
- **Multiplayer**: Colyseus / Supabase Realtime (w planach)
